# InvoiceVault

InvoiceVault merupakan aplikasi Cloud Invoice Manager berbasis PHP Native dan MySQL yang digunakan untuk mengelola arsip invoice dalam format PDF secara digital. Aplikasi ini memudahkan pengguna dalam menyimpan, mencari, melihat, mengunduh, dan menghapus invoice melalui antarmuka web yang sederhana dan responsif.

## Fitur

- Login Admin
- Upload Invoice PDF
- Preview Invoice
- Download Invoice
- Delete Invoice
- Search Invoice
- Filter berdasarkan bulan
- Dashboard Statistik
- Responsive UI

## Teknologi

- PHP Native
- MySQL
- Bootstrap 5
- JavaScript
- SweetAlert2
- HTML5
- CSS3

## Persyaratan

- PHP 8.x
- MySQL
- XAMPP
- Composer

## Cara Instalasi

1. Clone repository

```bash
git clone https://github.com/tamataaa/InvoiceVault.git
```

2. Pindahkan project ke folder `htdocs`.

3. Import database MySQL.

4. Jalankan Apache dan MySQL melalui XAMPP.

5. Buka browser.

```
http://localhost/InvoiceVault
```
